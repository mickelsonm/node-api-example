# node-api-example
Just a toy node api

### How do I run?

```node
  node index.js
```

### How do I see it in action?

Navigate to a web browser and put in the following URL(s):

* http://localhost:3000/users

* http://localhost:3000/users/external

> Note: The api must be running to see what these do!
